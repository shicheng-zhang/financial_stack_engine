RLAction get_rl_action(spread, vol, toxicity, rem_shares, time_rem) {
    // Rule 1: Time decay forces aggression
    if (time_rem < 0.2 && rem_shares > 0.1) return AGGRESSIVE_MARKET;

    // Rule 2: High queue toxicity (adverse selection) forces aggression
    if (toxicity > 0.7) return AGGRESSIVE_MARKET;

    // Rule 3: Wide spread + ample time = passive limit posting
    if (spread > 3.0 && time_rem > 0.5) return PASSIVE_LIMIT;

    // Default: Institutional VWAP behavior
    return PASSIVE_LIMIT;
}

The PPO agent learns these rules implicitly through reward shaping (-slippage). We extract a surrogate Decision Tree and compile it directly into the C++ hot path for nanosecond inference.
